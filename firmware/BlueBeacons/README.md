## 注意

* 请下载固件的原始格式，也就是通过raw链接下载
* 如果你是第一次升级，请下载imgB固件升级。反之则下载imgA固件。

## 已知问题

* 如果你使用iPhone 5S，按照升级步骤也可能升级不成功。请更换为iPhone5或者iTouch/iPad mini等其它设备尝试，含泪提醒各位。
